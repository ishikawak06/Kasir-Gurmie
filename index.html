☁️ Sync ke Google Spreadsheet
Masukkan Web App URL dari Google Apps Script. Data akan ditarik otomatis setiap load/transaksi jika Auto-sync aktif.
📖 Panduan Mendapatkan Web App URL
  1. Buka file Google Sheets Anda.
  2. Pilih menu Ekstensi > Apps Script.
  3. Klik tombol biru Terapkan (Deploy) di pojok kanan atas > pilih Deployment baru.
  4. Pilih jenis Aplikasi Web.
  5. Pada opsi Akses (Who has access), ubah menjadi "Siapa saja" (Anyone).
  6. Klik Terapkan, lalu Copy URL dan Paste di bawah ini.
